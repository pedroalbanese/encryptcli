Usage: EncryptCLI.exe -e|d --in <file.ext> --alg <algorithm> --key <key>

Commands:
   -e: Encrypt
   -d: Decrypt

Parameters:
   /alg: Algorithm
   /in : Input file
   /out: Output file (Optional [*])
   /key: Symmettric key

   [*] If no output is specified, the input file will be overwritten.

Algorithms:
   3DES, AES-128 (Default), AES-192, AES-256, DES, RC2, RC4
